sampatlh18 / pet.proj

This repository is a Java web application (Jakarta Servlets, JSP, MySQL). It is meant to run on a servlet container such as Apache Tomcat, not as a purely static website.

GitHub Pages only hosts static files (HTML, CSS, JavaScript). It cannot execute your .jsp pages or connect to MySQL the way your app does when you use Tomcat locally.

Run the project locally

  • Clone: git clone https://github.com/sampatlh18/pet.proj.git
  • Open the PetHubProject folder in Eclipse as a Dynamic Web Project (or build a WAR and deploy to Tomcat 10.x).
  • Import pethub.sql into MySQL and set env vars MYSQL_USER, MYSQL_PASSWORD, MYSQL_URL (see env.example).
  • Visit something like http://localhost:8080/PetHubProject/ after deployment.
